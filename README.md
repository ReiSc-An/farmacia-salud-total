# 💊 Farmacia Salud Total

## Descripción

Farmacia Salud Total es una aplicación web desarrollada con HTML, CSS y JavaScript que permite visualizar medicamentos, buscarlos y agregarlos a un carrito de compras.

## Funcionalidades

- Visualización de medicamentos.
- Buscador de medicamentos.
- Carrito de compras.
- Aumentar y disminuir cantidades.
- Eliminar productos.
- Cálculo automático del total.
- Formulario de contacto con validaciones.
- Diseño adaptable (Responsive).

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
