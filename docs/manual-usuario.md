# Manual de Usuario

## Inicio

Al ingresar al sistema se muestra la página principal de la Farmacia Salud Total.

## Buscar medicamentos

En la sección de productos existe un buscador donde el usuario puede escribir el nombre del medicamento para localizarlo rápidamente.

## Agregar al carrito

Presione el botón **Agregar al carrito** para añadir un medicamento.

## Modificar cantidades

Utilice los botones **+** y **-** para aumentar o disminuir la cantidad de un producto.

## Eliminar producto

Presione el botón **Eliminar** para quitar un medicamento del carrito.

## Contacto

Complete el formulario con su nombre, correo y mensaje para enviar una consulta.
